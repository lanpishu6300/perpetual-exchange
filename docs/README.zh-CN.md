# 文档

项目文档和归档文件。

[English](README.md) | [中文](README.zh-CN.md)

## 目录结构

```
docs/
├── README.md          # 本文件
└── archive/          # 归档的临时文档
```

## 文档说明

### 主要文档

项目的主要文档位于根目录：

- `README.md` - 项目主文档（英文）
- `README.zh-CN.md` - 项目主文档（中文）
- `ARCHITECTURE.md` - 架构设计文档
- `BENCHMARK_GUIDE.md` - 性能测试指南

### 压测报告

压测报告位于 `benchmark_reports/` 目录：

- `benchmark_reports/README.md` - 报告目录说明
- `benchmark_reports/CROSS_PLATFORM_BENCHMARK_REPORT.md` - 跨平台对比报告
- `benchmark_reports/mac/` - Mac平台报告
- `benchmark_reports/docker/` - Docker平台报告

### 归档文档

`archive/` 目录包含开发过程中的临时文档和状态报告，已不再维护，但保留作为历史参考。

## 语言支持

- **英文**: 代码和文档的主要语言
- **中文（简体）**: 文档的辅助语言

所有关键文档都提供英文和中文版本。
