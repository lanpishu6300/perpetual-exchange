# Documentation

Project documentation and archived files.

[中文](README.zh-CN.md) | [English](README.md)

## Directory Structure

```
docs/
├── README.md          # This file
└── archive/          # Archived temporary documents
```

## Documentation

### Main Documentation

Main project documentation is located in the root directory:

- `README.md` - Main project documentation (English)
- `README.zh-CN.md` - Main project documentation (Chinese)
- `ARCHITECTURE.md` - Architecture design document
- `BENCHMARK_GUIDE.md` - Performance testing guide

### Benchmark Reports

Benchmark reports are located in `benchmark_reports/` directory:

- `benchmark_reports/README.md` - Reports directory guide
- `benchmark_reports/CROSS_PLATFORM_BENCHMARK_REPORT.md` - Cross-platform comparison report
- `benchmark_reports/mac/` - Mac platform reports
- `benchmark_reports/docker/` - Docker platform reports

### Archived Documents

The `archive/` directory contains temporary documents and status reports from the development process. These documents are no longer maintained but are kept for historical reference.

## Language Support

- **English**: Primary language for code and documentation
- **Chinese (Simplified)**: Secondary language for documentation

All key documents have both English and Chinese versions.
